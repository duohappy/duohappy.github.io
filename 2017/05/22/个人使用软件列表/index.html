个人使用软件列表

发表于 2017-05-22

系统安全类

  • Win10
    • 使用bash
  • Virtualbox
    • Ubuntu
  • Sandboxie
    • 有些网站，嗯，你懂的
  • eset
    • 防火墙很好用
  • TC
    • 不一定需要使用快捷键，它的双窗口本身就很有用，只用鼠标点击也可以达到效率提升的目的。不要给自己设限
  • cmder
    • Windows下，Python有时会提示编码错误的问题，使用cmder运行py文件试试看（前提配置好cmder）
  • gitbash
    • 很好用的git工具
  • github desktop
    • 一开始学习git的时候，使用的是命令行，操作github也是使用命令行。现在几乎都是用图形界面操作。图形界面让问题变得简单，如果对github的操作不是太复杂，那么推荐使用github desktop（真心不好安装，要做好心理准备）

文本编辑类

  • 不推荐使用Notepad++
  • EmEditor
    • 编辑csv真是一绝
  • 国产EverEdit
    • 打开大文件很快
    • 界面漂亮
  • 写Python有时候需要调试，会使用VSCode
  • texlive
    • 有时会写一些TeX的文档，虽然很少
  • Linux下，gedit、kate、Vim等

数据可视化

  • Excel
    • Power Query
    • Power Pivot
    • Power View（不看好这个插件）
  • Tableau
  • PowerBI
  • Python中的matplotlib

集成开发环境

  • Python
    • PyCharm
  • Java
    • Eclipse
    • 面向对象的思想，还真是学习了Java之后才明白的（封装、继承、多态），我是学习了Java后才学的Python。
  • matlab
    • matlab，:)
  • C
    • CodeBlocks
  • C#
    • VS
  • VB
    • 我学的时候，学校给的教材还是VB 6.0

知识管理类

  • 为知笔记
    • 用了近5年，很好用

影音图像

  • potplayer
    • 简洁大方好用
  • IrfanView
    • 快
  • paint.net
    • 伪资深PS高手的替代品

科研工具

  • Axmath
    • 足以代替MathType的国产公式编辑器
    • 价格极其实惠
  • Zotero
    • 开源文献管理工具，和本土的NoteExpress可以互补
