安全菜鸟如何防范勒索病毒

发表于 2017-05-22

如果你和我一样是安全方面的菜鸟，那么这篇文章应该有点借鉴意义，本文也仅仅针对像我一样的安全菜鸟，且仅仅针对于windows系统。

写在前面

以前网上流传一种说法，大意是安装原装windows系统，关闭windows自动更新，不装杀毒软件，完完全全裸奔。这种说法受到了很多人追捧，其中就一句话说对了，安装原装windows系统。

作为安全菜鸟的大多数，最好了解基本的安全软件，并且用起来。别人说的话能防毒么？

系统

正版系统

不要用windows xp，不要用微软不提供安全补丁的windows系统。2020年，win7也要寿终正寝

我在三年前购买了正版win8，一路更新过来，现在免费更新到了win10。三年了，没有重装过一次系统

用户帐户控制设置

我一般都会调到最高

自动更新

现在很多人都是正版的win10系统了，不要把盗版的习惯一直延续下去，务必打开自动更新

安全软件

杀毒软件

我目前用的是正版eset，图标还蛮酷的

关于防火墙，我用的是eset防火墙中的交互模式，刚开始所有联网的进程都必须弹窗提醒

  • 其他杀毒软件：
    • AVG（好像被收购了，我用过很长一段时间）
    • Avast（也用过一段时间）
    • Windows Defender（用的时间不长）
    • Avira （口碑不错）
    • BitDefender （口碑不错）
    • Kaspersky （口碑不错）
    • …….

提一个保守的做法：尽可能地信任杀毒软件的报毒，自己拿不准的报毒，一律相信杀毒软件的判断。至于涉及误报的问题，这个要讲的话就不是一篇文章的事情了，可以google一下

沙盘

免费的版本就够用，不是所有的软件都要去破解一下，破解的补丁搞不好还有毒呢。只要是拿不准的软件或者网址，一定入沙。

虚拟机

  • virtualbox（开源免费）

拿不准的软件或者网址也可以入虚拟机，善用快照功能，我目前装了一个ubuntu系统（为了学习多进程之fork），最多的时候还装了win7和winxp

  • VMware
    很流行，还没有用过
